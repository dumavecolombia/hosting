¿Qué es HTML?

HTML (HyperText Markup Language) es el lenguaje que se usa para crear la estructura de las páginas web. Permite agregar títulos, textos, imágenes, enlaces y formularios.

Es el primer lenguaje que todo programador web debe aprender.

¿Qué es CSS?

CSS (Cascading Style Sheets) se utiliza para dar estilo a las páginas web creadas con HTML. Permite cambiar colores, tamaños, márgenes, fuentes y diseños.

Gracias a CSS, una página puede verse bonita, ordenada y profesional.

¿Qué es un Hosting?

Un hosting es un servicio que permite guardar tu página web en un servidor para que cualquier persona pueda verla desde internet.

Sin hosting, tu página solo existiría en tu computadora.
