¿Qué es HTML?

HTML es el lenguaje que define la estructura de una página web. Permite crear títulos, párrafos, imágenes, enlaces y formularios.

<h1>Hola Mundo</h1> <p>Mi primera página web</p>

¿Qué es CSS?

CSS se usa para dar color, tamaño y diseño visual a las páginas creadas con HTML.

h1 { color: blue; } body { background: lightgray; }
