¿Qué es HTML?

                      

HTML (HyperText Markup Language) es el lenguaje estándar utilizado para crear y estructurar páginas web. Es la base fundamental de cualquier sitio en Internet y permite definir la organización y el contenido que se muestra en los navegadores.

HTML no es un lenguaje de programación, sino un **lenguaje de marcado**, lo que significa que utiliza etiquetas para describir y estructurar el contenido, como títulos, párrafos, imágenes, enlaces, tablas y formularios.

La versión moderna del estándar es **HTML5**, que incorpora nuevas etiquetas semánticas, soporte multimedia nativo y mejoras en rendimiento y compatibilidad.

---

### 2. ¿Para qué sirve HTML?

HTML permite:

* Estructurar el contenido de una página web.
* Insertar textos, imágenes, videos y enlaces.
* Crear formularios para recopilar datos del usuario.
* Organizar información mediante listas y tablas.
* Definir la base sobre la cual trabajan CSS y JavaScript.

HTML trabaja junto con:

* **CSS**: para el diseño y estilos visuales.
* **JavaScript**: para la interactividad y dinamismo.

---

### 3. ¿Cómo funciona HTML?

HTML utiliza **etiquetas (tags)** que indican al navegador cómo debe mostrarse el contenido. Estas etiquetas generalmente se escriben entre signos de menor y mayor que:

```html

```

* `

* `
